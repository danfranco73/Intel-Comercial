Trabajá sobre la base comercial para consultar indicadores, filtros, rankings, insights y análisis dinámicos sin exponerte a tareas operativas del sistema.

Uso recomendado

  1. Elegí el período del informe.
  2. Definí el enfoque comercial por agrupaciones.
  3. Actualizá el BI con el botón principal.
  4. Refiná métricas y rankings con filtros del informe.
